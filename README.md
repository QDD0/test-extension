# Разработка расширения для мониторинга активности пользователя в браузере при прохождение тестирования

## Описание проекта
Система для проведения контролируемого онлайн-тестирования с отслеживанием активности пользователя в браузере. Включает веб-приложение для тестирования и браузерное расширение для мониторинга нарушений.

## 💻 Технологический стек
Frontend:
- Angular 17+ - фреймворк для SPA
- TypeScript - язык программирования
- HTML5/CSS3 - верстка и стили

Backend:
- Java 17 - язык программирования
- Spring Boot 3 - фреймворк
- Spring Security - безопасность
- Spring Data JPA - работа с БД
- PostgreSQL - база данных
- JWT - аутентификация

## 🚀 Как запустить

### База данных
1. Настройте подключение в файле `compose.yaml` (порт, логин, пароль)
2. Проверьте настройки в `application.properties`
3. Создайте таблицы, выполнив скрипт из `tables.sql`
4. **Важно**: Сначала зарегистрируйте пользователя через интерфейс, затем выполните `data.sql` для заполнения тестовыми данными

### Frontend
- Используется ***Angular***
- Убедитесь, что есть ***Node.js*** и ***npm*** (чтобы проверить - ```ng version```) 
- Сначала переходим в ```cd \frontend``` и прописываем ```ng serve``` для запуска и переходим в <http://localhost:4200/>

### Backend
Чтобы запустить полноценно, нужно выполнить указания с бд, а также понадобиться **jwtSecret** (для примера: ```ThisIsAVeryLongSecretKeyThatIsAtLeastSixtyFourCharactersLongForHS512AlgorithmSecurity987654321```). Остальное разбито по пакетам 

### Расширение
Находится в папке extension, в основном с ним делать ничего не надо. Для добавления его в браузер, заходим в расширения, выбираем 
загрузить распакованное расширение и указываем папку extension. Без расширения, не запускаются сами тесты.
